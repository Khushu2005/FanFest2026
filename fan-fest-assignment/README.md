# FanFest 2026 UI Replication

This project is a high-fidelity, fully responsive UI replication of the FanFest 2026 landing page.

## Tech Stack Used
- React (Vite)
- Pure SCSS (Mobile-First Architecture)
- Framer Motion (Scroll Animations)

## Quick Setup
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the local development server

## Key Features
- Pixel-perfect typography and grid matching.
- Exact dual-glow background replication.
- Fluid responsiveness using CSS clamps and grid layouts.
